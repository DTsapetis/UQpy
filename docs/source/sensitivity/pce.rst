PCE Sensitivity
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^


The :class:`.PceSensitivity` class is imported using the following command:

In order to use the :class:`.PceSensitivity` class, the user must previously train a :class:`.PolynomialChaosExpansion`
surrogate which will then be used as an input argument to the initializer of the following sensitivity class.
For examples regarding the usage of the :class:`.PceSensitivity` class please refer to :class:`.PolynomialChaosExpansion`

>>> from UQpy.sensitivity.PceSensitivity import PceSensitivity
